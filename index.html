🔗 Syllogisms

  • All A are B + All B are C → All A are C ✅
  • All A are B + Some B are C → Some A are C ❌ (not always)
  • No A is B + All C are A → No C is B ✅
  • Some A are B + All B are C → Some A are C ✅
  • Always check for "possibility" conclusions — they differ from definite ones
💡 Draw Venn diagrams mentally for every syllogism question

🔐 Coding–Decoding

  • Letter shift: find pattern (+2, −3, alternate, reverse alphabet)
  • Reverse alphabet: A=26, B=25... Z=1
  • Number-letter mapping: A=1, B=2, … Z=26
  • Mixed coding: break word into pairs or triplets and look for pattern
  • Conditional coding: match conditions to assign digits/symbols

👨‍👩‍👧‍👦 Blood Relations

  • Father's/Mother's son/daughter = sibling
  • Spouse's father/mother = father-in-law / mother-in-law
  • Son's/Daughter's child = grandson/granddaughter
  • Draw a family tree diagram on rough paper
  • Watch for gender clues: "he/she" helps determine generation

🧭 Direction Sense

  • Start from initial position; track each turn (left/right)
  • After facing North: Left=West, Right=East
  • Displacement = √(x² + y²) using Pythagoras
  • Shadow falls opposite to sun direction (morning sun = East, shadow = West)

🔄 Series & Patterns

  • Check: arithmetic (+n), geometric (×n), square/cube series
  • Fibonacci-type: each term = sum of previous two
  • Alternating series: two interleaved sequences
  • Difference of differences: 2nd level differences may be constant
  • Letter series: track position jumps (+2, −1, etc.)

💺 Seating & Arrangement

  • Linear: note exact positions, use elimination method
  • Circular: fix one person, arrange remaining (n−1)!
  • Facing center vs outside: affects left/right orientation
  • Floor puzzles: start from definite clues (top/bottom floors)

📚 Reading Comprehension

  • Read the questions first, then skim the passage
  • Main idea = usually in first and last paragraphs
  • "Author's tone" options: critical, neutral, appreciative, sarcastic
  • Inference Qs: answer must be supported by the passage, not outside knowledge
  • Watch for "EXCEPT" and "NOT" questions — easy traps

✏️ Grammar — Common Rules

  • Subject-verb agreement: singular subject → singular verb
  • "Either/Neither" takes singular verb when both subjects are singular
  • Tense consistency: don't mix past and present in the same sentence
  • Prepositions: "interested in", "good at", "different from", "similar to"
  • Articles: use "an" before vowel sounds (an hour, an MBA)

🔀 Para Jumbles

  • Find the opening sentence — usually introduces a topic without pronouns
  • Find the closing sentence — concludes/summarizes
  • Link sentences using pronoun references (it, they, this, that)
  • Connectives signal order: "however", "therefore", "moreover", "but"

💬 Vocabulary Tips

  • Synonyms/antonyms: look for root words (Latin/Greek roots)
  • Analogy: identify relationship type (cause:effect, part:whole, action:doer)
  • Idioms: context determines meaning; memorize common ones
  • Cloze test: read full paragraph first, then fill blanks logically

📊 Bar & Line Charts

  • Always read axis labels and units first
  • % change = (new−old)/old × 100
  • Ratio comparison: cross multiply to avoid decimals
  • Approximation is key — don't compute exact values when options are far apart

🥧 Pie Charts

  • Sector value = (degree/360) × total OR (percentage/100) × total
  • Difference in sectors = difference in their percentages × total
  • When total is not given, only ratios can be found

🗃️ Tables & Caselets

  • Identify which rows/columns are needed — don't read everything
  • Calculate row/column totals when needed for percentage questions
  • Caselets: convert text into a table/diagram mentally first
  • Use approximation: 48% ≈ 50%, 33% ≈ 1/3 etc.
